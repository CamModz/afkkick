INSTALLATION:

1. Rename the file too afkkick ( remove the -master )
2. Place in fiveM scripts folder or wherever you put scripts.
3. ensure afkkick ( In server.cfg )
4. Make sure the resource is found in your txAdmin or is started ( Just restart the server once added )
5. Enjoy!
